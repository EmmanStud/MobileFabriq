## ✨ Key Interactions

1. **Typing Contact Number**
   - User types: "9171234567"
   - System shows: "+63 9171234567"
   - Field refuses more than 10 digits
   - Automatically formatted

2. **Selecting Dates**
   - Start Date: Tap to open calendar
   - Cannot select past dates
   - End Date: Tap to open calendar
   - Cannot select before start date

3. **Changing Gown**
   - Select different gown
   - Summary updates immediately
   - New price reflected
   - Calculations recalculate

4. **Summary Updates**
   - Every field change triggers recalculation
   - Total = Duration × Daily Rate
   - Downpayment = Total ÷ 2
   - Instant, no delay
